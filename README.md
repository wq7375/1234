这是一个简单易用的课程预约小程序，专为课程管理和预约设计，无需编程经验即可快速部署和使用。

## 主要功能

### 👤 用户功能

- **账号登录**：使用微信一键登录，自动区分管理员和普通用户
- **课程预约**：浏览可预约课程，选择合适的时间进行预约
- **预约管理**：查看个人预约历史，取消已预约课程
- **卡券管理**：查看课程卡券余额和有效期

### 🔧 管理员功能

- **课程管理**：创建和编辑课程信息，设置课程时间和容量
- **课表管理**：管理每周课表，支持复制上周课表快速创建
- **用户管理**：查看用户列表和预约情况
- **自动取消**（实现中）：系统自动取消人数不足的课程并通知用户

## 快速开始

### 准备工作

1. 注册[微信公众平台](https://mp.weixin.qq.com/)账号
2. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 部署步骤

1. 克隆或下载本项目代码到本地
2. 打开微信开发者工具，导入项目
3. 在项目设置中填写你的小程序AppID
4. 点击"云开发"按钮，开通云开发环境
5. 上传所有云函数（cloudfunctions目录下所有文件夹）
6. 点击"编译"按钮运行小程序

## 使用说明

### 普通用户

1. 打开小程序后自动登录
2. 在"约课"页面浏览可预约课程
3. 选择课程点击"预约"按钮完成预约
4. 在"我的"页面查看预约历史和卡券信息

### 管理员

1. 使用管理员账号登录（系统中第一个注册的账号默认为管理员）
2. 进入"管理"页面进行课程和用户管理
3. 设置每周课表和课程信息
4. 查看课程预约情况和用户数据

## 常见问题 (FAQ)

### 账号与权限

**Q: 如何成为管理员？**
A: 系统中第一个注册的用户会自动成为管理员，拥有课程管理和用户管理权限。

**Q: 忘记登录信息怎么办？**
A: 小程序使用微信一键登录，无需额外密码。确保使用注册时的微信账号即可正常登录。

### 课程预约

**Q: 为什么无法预约某些课程？**
A: 可能原因包括：课程人数已满、所选课程卡券与课程类型不匹配（私教卡只能预约私教课）、或课程已开始。

**Q: 如何取消预约？**
A: 在"我的"页面找到对应预约记录，点击"取消预约"。注意：部分课程可能有取消时间限制。

**Q: 课程被取消会收到通知吗？**
A: 暂时不是，当课程因人数不足被系统自动取消时，已预约用户暂时不会收到微信通知，同时课程次数会自动返还到您的卡券中。（相关功能实现中）

### 卡券与次数

**Q: 如何查看我的课程次数？**
A: 在"我的"页面可以查看所有卡券余额和有效期。

**Q: 课程取消后次数会返还吗？**
A: 是的，系统取消或提前取消预约都会自动返还课程次数到原卡券。

### 系统功能

**Q: 为什么有些课程会被自动取消？**
A: 当课程预约人数未达到最低开课人数（在课程设置中定义），且距离开课时间不足2小时时，系统会自动取消课程。

**Q: 如何复制上周课表？**
A: 管理员在"课表管理"页面点击"复制上周课表"，可快速创建本周课表。
